# Welcome to YelpCamp!
![image](https://github.com/user-attachments/assets/7355f9f7-1fa4-4ad6-84cb-95c7c04ca5c2)


### YelpCamp is a personal, FullStack project featuring a login system, where you, the user, can browse and add locations to the world map, which is updated in real time, and everyone can see and review any locations they wish.
